Mes projets

Apprenti Ordonnateur

Situation : Dans le royaume de Krystalia, les cristaux des temples ont changé de couleur après une tempête. L'apprenti ordonnateur doit réaligner les cristaux dans les temples.

Tâche : Créer un programme pour réaligner les cristaux de manière optimale, avec une interface graphique permettant de visualiser le processus.

Action : Développement d'un algorithme de déplacement de l'apprenti pour échanger les cristaux, avec plusieurs algorithmes de tri (sélection, bulle, rapide). Création d'une interface graphique avec JavaFX pour afficher les temples, cristaux, et le nombre de déplacements.

Résultat : Le programme fonctionne, permettant à l'apprenti de réaligner les cristaux efficacement. L'interface graphique est interactive et affiche le nombre de déplacements réalisés.

Monitoring Hardware avec Prometheus & Grafana

Situation : L'entreprise avait besoin d'une solution centralisée pour surveiller l'état du matériel (CPU, RAM, stockage, température, réseau).

Tâche : Concevoir et déployer une solution de supervision complète en utilisant Prometheus pour la collecte des métriques et Grafana pour la visualisation.

Action : Développement d'un exporter personnalisé pour récupérer les métriques hardware, configuration des dashboards Grafana, mise en place d'alertes et intégration dans l'infrastructure existante.

Résultat : Une plateforme de monitoring fiable permettant aux équipes techniques de suivre en temps réel la santé des serveurs et d'anticiper les incidents.


Mes expériences

Stage – Projet de Monitoring Hardware (Prometheus & Grafana)

Entreprise : Evermedia

Durée : 8 semaines

Contexte : L'entreprise souhaitait améliorer la surveillance de ses serveurs en centralisant les informations matérielles essentielles : CPU, RAM, état du SmartStorage, charge réseau, température des composants (CPU, inlet, ambient, chipset, BMC), etc.

Objectif : Concevoir une solution complète de supervision permettant d’anticiper les défaillances matérielles et d’optimiser la visibilité sur l’état des serveurs.

Réalisations :

  • Création d’un exporter personnalisé pour extraire les métriques hardware non disponibles dans les outils classiques.
  • Collecte automatique de données : utilisation CPU/RAM, activité réseau, statut RAID, batterie SmartStorage, températures internes détaillées.
  • Installation et configuration de Prometheus pour la collecte continue des métriques.
  • Conception de dashboards Grafana avancés pour une visualisation claire en temps réel.
  • Configuration d’un système d’alerting (seuils de température, anomalies réseau, surcharge CPU/RAM, erreurs SmartStorage).
  • Rédaction d’une documentation technique complète pour assurer la continuité du projet.

Résultat : Une plateforme de monitoring fiable, personnalisée et facilement exploitable par l’équipe technique, permettant une meilleure détection des anomalies matérielles.
